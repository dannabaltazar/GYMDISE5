Modelo de Comportamiento

Nivel 2

Descripción

El modelo de comportamiento del sistema Body Xtreme se divide en múltiples niveles jerárquicos:

Componentes Principales:

  • Proceso de Inscripción: Captura de datos de miembros, validación y generación de membresías
  • Proceso de Pagos: Integración con pasarelas de pago QR y tarjeta de crédito
  • Proceso de Gestión de Horarios: Asignación de instructores a sucursales y franjas horarias
  • Proceso de Consultas: Validación de membresías, renovaciones y acceso a sucursales
  • Proceso de Reportes: Generación de informes para administración e instructores

Almacenes de Datos:

  • Base de Datos de Miembros: Información personal, membresías activas y historial
  • Base de Datos de Pagos: Transacciones, métodos de pago y comprobantes
  • Base de Datos de Sucursales: Ubicaciones, horarios e instructores asignados
  • Base de Datos de Instructores: Datos laborales, especialidades y asignaciones
