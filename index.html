GENERALIDADES

En esta web se puede observar el análisis y diseño de un sistema web para el gimnasio Body Xtreme, cuyo propósito es optimizar el proceso de registro, control y seguimiento de membresías mediante la implementación de una plataforma virtual. Este sistema permitirá a los usuarios registrarse en línea, realizar pagos de manera digital, consultar información sobre sucursales y agendar consultas con especialistas. Además, brindará herramientas a la administración e instructores para gestionar información de miembros, horarios, reportes e ingresos de forma eficiente. Con esta iniciativa, se busca integrar la gestión administrativa y la experiencia del usuario en un entorno moderno, seguro y accesible desde cualquier dispositivo conectado a internet.

En la actualidad, la mayoría de gimnasios continúan utilizando sistemas manuales o herramientas no integradas para el manejo de inscripciones, pagos y control de membresías. Esta situación genera pérdida de tiempo, errores humanos y dificultades en el seguimiento de los clientes activos. Body Xtreme ha identificado la necesidad de evolucionar hacia la digitalización de sus procesos administrativos y de atención al cliente. El crecimiento de la demanda de usuarios y la expansión de sucursales exigen un sistema eficiente que permita centralizar la información y automatizar tareas como la renovación de membresías, generación de reportes y control de instructores.

El gimnasio Body Xtreme enfrenta dificultades en la gestión y control de sus miembros debido a la ausencia de un sistema automatizado que centralice la información. Actualmente, los procesos de inscripción, pagos, control de vencimientos y reportes se realizan de manera manual, lo cual provoca errores frecuentes, demoras en los trámites y pérdida de datos importantes. Asimismo, los usuarios no cuentan con una plataforma virtual donde puedan registrarse, pagar sus membresías o consultar información sobre servicios y sucursales. Esta falta de digitalización limita la accesibilidad y reduce la competitividad frente a otros gimnasios que ya ofrecen servicios en línea.

HOJAS (Efectos)
TRONCO (Problema central)
RAÍCES (Causas)

¿Cómo puede el gimnasio Body Xtreme mejorar la gestión de sus membresías, pagos, instructores y reportes mediante la implementación de un sistema web que automatice estos procesos y facilite la inscripción y pago en línea de sus miembros?

Diseñar e implementar un sistema web de registro y gestión de membresías para el gimnasio Body Xtreme que permita optimizar los procesos de inscripción, pago y control administrativo, garantizando mayor eficiencia, accesibilidad y satisfacción del cliente.

OBJETVO GENERAL:

Desarrollar un sistema de registro y gestión de miembros en línea para el gimnasio Body Xtreme, que permita realizar inscripciones, pagos digitales, control de membresías y generación de reportes administrativos de manera automatizada.

OBJETIVOS ESPECIFICOS:

1. Analizar los procesos actuales de registro y control de membresías en el gimnasio Body Xtreme.

2. Diseñar una interfaz web funcional y accesible para el registro en línea de nuevos miembros.

3. Implementar un módulo de pagos digitales mediante código QR o tarjeta de crédito.

4. Integrar un sistema de control de membresías que permita la gestión de vencimientos y renovaciones automáticas.

5. Desarrollar un módulo administrativo que genere reportes sobre ingresos, miembros activos y expirados.

6. Permitir a los instructores consultar su horario y reportes de pago de manera individual.

Descripción - Modelo Ambiental

El Modelo Ambiental es un diagrama que muestra el contexto del sistema de registro del gimnasio Body Xtreme. Incluye:

  • Actores externos: Miembros, Administradores, Instructores
  • Factores ambientales: Sistema de pagos, Base de datos, Servidores
  • Interfaces: Cómo interactúan los actores con el sistema

Este diagrama es fundamental para entender cómo el sistema se relaciona con su entorno y qué elementos externos lo afectan.

Ver Diagrama Completo
Descripción - Modelo de Comportamiento

El Modelo de Comportamiento describe cómo el sistema Body Xtreme funciona dinámicamente a través del tiempo:

  • Procesos: Registro, Pago, Renovación de membresía
  • Flujos de datos: Cómo circula la información en el sistema
  • Estados: Diferentes estados de una membresía (Activa, Vencida, Cancelada)

Este diagrama es esencial para comprender la lógica operativa y el flujo de información del sistema.

Ver Diagrama Completo
Descripción - Diagrama Orientado a Objetos

El Diagrama Orientado a Objetos muestra la estructura de clases del sistema:

  • Clase Miembro: Atributos y métodos de los usuarios del gimnasio
  • Clase Membresía: Gestión de paquetes y vencimientos
  • Clase Instructor: Información y horarios de instructores
  • Clase Pago: Registro de transacciones

Las relaciones entre clases muestran cómo los objetos interactúan entre sí.

Ver Diagrama Completo
Descripción - Casos de Uso

El Diagrama de Casos de Uso describe las interacciones entre actores y el sistema:

  • Casos del Miembro: Registrarse, Pagar membresía, Ver información
  • Casos del Administrador: Gestionar miembros, Ver reportes, Administrar instructores
  • Casos del Instructor: Ver horario, Reportar asistencia, Ver estudiantes

Este diagrama es útil para entender qué acciones puede realizar cada tipo de usuario.

Ver Diagrama Completo

Descripción - Modelo SCRUM

SCRUM es la metodología ágil utilizada para gestionar el proyecto:

  • Sprints: Ciclos cortos de desarrollo (2-4 semanas)
  • Product Backlog: Lista de funcionalidades a implementar
  • Daily Standup: Reuniones diarias de coordinación
  • Retrospectivas: Análisis de mejora continua

SCRUM permite entregas frecuentes, adaptabilidad y comunicación constante con stakeholders.
